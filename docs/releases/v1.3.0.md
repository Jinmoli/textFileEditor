## GitHub Release 正文

```markdown
## 本次更新

- 新增编辑器行号显示能力，支持跟随浅色/深色主题自动适配。
- 当前光标所在行号会同步高亮，阅读与定位更直观。
- 设置页新增“显示行号”开关，默认开启，并可立即作用到已打开的编辑器。
- 优化 `只读` 与 `自动换行` 按钮的激活态样式，降低暗色主题下的刺眼感，提升图标与文字辨识度。

## 说明

这是一个面向编辑体验的功能更新版本，适合希望在 Obsidian 中获得更稳定行号显示和更柔和工具栏状态反馈的用户升级使用。
```
